⚠️ BEFORE DEPLOYING

1. Change SECRET_KEY in .env
2. Set GOOGLE_PLACES_API_KEY in .env
3. Update admin password via API or SQL
4. Configure CORS for production
5. Set DEBUG=False in .env
6. Test all API endpoints
7. Build frontend React components
8. Set up HTTPS certificates
9. Configure database backups
10. Set up monitoring/logging
